# Math

This repository contains a simple index.html page and an accompanying minimal server implementation for rendering it.

## Simple Render Server

To launch the server using Node.js and Express:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

3. Visit `http://localhost:3000` in your browser to view `index.html`.

*Alternative:* run a quick static file server with Python:

```bash
cd /workspaces/Math
python3 -m http.server 8000
```
